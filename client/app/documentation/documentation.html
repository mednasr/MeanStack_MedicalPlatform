Installation

Just 3 steps to install the application in your local machine.

Step-1:

Download, Install NodeJS (Acts as our Server)
Download, Install MongoDB (The Database for our application). We also need to run mongodb. Browse to the (MongoDB installation folder)/bin and run mongod

Step-2:

Unzip the downloaded file

Step-3:

Windows Double click on start.bat found inside the directory
Now your browser should automatically open up with the application running on http://localhost:8080
Linux Open your terminal, navigate to the extracted directory and enter the command node .
Open your browser and navigate to http://localhost:8080

If you face any issue, I will do it for you in your machine or server

There are web hosting solutions which offers free hosting for this APP. (e.g. Openshift, Amazon web services, Heroku)

Medical

This view displays all Clinics with following facilities

  • Clinicsearch
  • Filter by Type, Travel_Agenciesry, price (with price slider)
  • Sort Clinics by price or Clinicname
  • Add to
  • Checkout with paypal
  • Auto paging facility which activates on scroll (Like most of the advanced websites)

::::::::::::::: MedicalAdministration ::::::::::::::

Only administrators can access the pages

Manage Types

Administrators can add, edit, delete, filter Types of their Medicalfrom this view

Manage Clinics

This is the main page for administrators to manage Clinics at Medical.

  • The right sidebar lists all the available Clinics with a search box to filter the list.
  • Clicking on a Clinicat the Cliniclist will populate the details of the Clinicat the left sidebar
  • The left sidebar has option to change Clinicname, details, Type, Travel_Agenciesry
  • This left sidebar also contains a module to manage Clinicvariants which has facility for Size, MRP, Price and Image for that perticular variant

Manage Users (Customers)

Using this view administrators can add, remove or edit users of their shopping web application
